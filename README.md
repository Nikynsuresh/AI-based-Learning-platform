🚀 AI-Based Personalized Learning Platform

An AI-powered adaptive learning platform that understands a student’s strengths and weaknesses to generate personalized explanations, quizzes, and study plans.

Built using FastAPI, React, MongoDB, and Google Gemini, the system combines Retrieval-Augmented Generation (RAG), PDF knowledge extraction, and web scraping to create a smart AI tutor that evolves with the learner.

⸻

✨ Features

🤖 AI Tutor

Ask questions and receive clear, personalized explanations generated by AI.

📚 Personalized Learning Paths

The system analyzes learning behavior and automatically adapts study plans based on weak areas.

📄 PDF Knowledge Integration

Upload textbooks or notes and allow the AI to answer questions directly from your documents.

🔎 Retrieval-Augmented Generation (RAG)

Combines document retrieval + LLM intelligence to generate accurate, contextual responses.

🌐 Educational Web Scraping

Collects relevant learning materials from the web to enhance AI responses.

🧪 Auto Quiz Generation

Generates practice quizzes to reinforce concepts.

📊 Learning Progress Tracking

Stores learning history and identifies knowledge gaps.

🔐 Secure Authentication

JWT-based authentication with encrypted password storage.

⚙️ Workflow

1️⃣ Authentication
	•	User signs up or logs in
	•	Password verified using Bcrypt
	•	JWT token issued for secure sessions

2️⃣ AI Learning
	1.	User asks a question
	2.	RAG retrieves relevant knowledge
	3.	Gemini AI generates explanations or quizzes
	4.	Interaction stored to improve learning insights

3️⃣ Knowledge Integration
	•	Upload PDFs
	•	Extract text using PyPDF
	•	Index content for RAG retrieval
	•	AI answers questions from uploaded materials
